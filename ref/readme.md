## Regenerating Markdown Files

To regenerate the markdown files from the PDFs:

```bash
pip install pdfplumber
python ref/convert_pdfs.py
```

To convert a single section:

```bash
python ref/convert_pdfs.py --pdf "RB209_Section7_2020_200113_WEB.pdf"
```

To verify conversion quality:

```bash
python ref/convert_pdfs.py --verify
```
